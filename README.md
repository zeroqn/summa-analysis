# Summa Solvency

A monorepo for Summa, zk proof of solvency protocol

### Subdirectories

- `zk_prover`: Halo2 based zk prover for Summa
- `contracts`: Solidity smart contracts for Summa
- `backend` : Rust API to interact with Summa
